# Future Supabase Integration

Estos archivos están pausados mientras LearnQuest funciona en modo local-first.

- `supabase.ts.disabled`: cliente real y mock anterior de Supabase.
- `AuthForm.tsx.disabled`: formulario de autenticación para una futura fase con login.
- `schema.sql`: esquema PostgreSQL/Supabase planificado.
- `.env.local.example`: variables públicas que se necesitarán cuando se reactive Supabase.

No importes estos archivos desde `src/` durante la demo local.
